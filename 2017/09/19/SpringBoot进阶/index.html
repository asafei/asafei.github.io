  • AOP是一种编程范式
    • 与语言无关，是一种程序设计思想
    • 面向切面（AOP）Aspect Oriented Programming
    • 面向对象（OOP）Object Oriented Programming
    • 面向过程（POP）Procedure Oriented Programming

AOP将通用逻辑从业务逻辑中分离出来。
